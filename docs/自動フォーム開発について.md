# 自動フォーム開発について  

## アプリと機能登録について 

- master_schema.mtb_item_categoryにアプリと機能を登録   
  - category_code: アプリの記号
  - category_name: アプリの名称
  - sub_category_code: 機能
  - remarks: 説明
  - data_structure: single/repeatable/page.
    - single: 自動生成単一のフォーム。例えば、ユーザー不情報フォーム
    - repeatable: 自動生成重複フォーム。例えば、経歴とか
    - page: 自動生成しないマニュアルフォームページ
  - enabled: 有効無効。active/inactive。提供したくない情報種類をinactiveにする  
  - show_order: tabとして表示順番
  - ui_component: コンポネント。


## アプリの機能の項目登録について  

- master_schema.mtb_item_dictionaryにアプリの機能の項目登録を登録
  - g_item_code: 項目のグローバルユニック記号
  - l_item_code: 項目の機能内のローカルユニック記号
  - category_code: アプリ記号
  - sub_category_code: 機能記号
  - field_definition: フィールドの詳細定義。項目の編集関連入力com(v-inputとか)、編集可能かどうか、数値か、文字化、金か、最小、最大値など情報。
  - formula: 計算項目の場合PG計算式を記入
  - enabled: 有効無効。active/inactive。表示たくない情報はinactiveにする
  - cal_order: 計算項目の場合計算順番
  - showable: 項目を表示するかどうか(show/hide)。裏で計算用、あるいは提供したくない場合はhideにする
  - show_order: 表示順番
  - item_description: 項目説明、範囲、値の一覧（選択の場合）(未使用)

## 項目field_definitionの説明  

```json
{
  # フォーム内部キー  
  "key": "staff_name",
  # DB情報 
  "db": {
    "column": "staff_name",
    "dataType": "varchar"
  },
  #  表示コンポーネント。自分定義コンポーネント可。例："component":"v-text-field", "component":"PermissionRuleEditor"
  "component": "v-text-field",
  # 表示名
  "label": "氏名",
  # データ型。text,number,date,datetime,boolean,email,tel,json,arrayなど
  "type": "text",
  # 初期値
  "default": null,
  #  必須(true/false)
  "required": true,
  # 編集不可(true/false)
  "readonly": false,
  # 非表示(true/false)
  "hidden": false,
  # Vuetify Props。
  /* 
    { "props":{ "clearable":true, "variant":"outlined" }}。
    上記componentによって合わせてそれぞれ違う内容を定義できます
  */
  "props": {},
  # 検証。
  /*
        rule: {}でも可。例：{ "validation":{ "minLength":2,"maxLength":50 }}
  */
  "validation": {},
  # 画面表示制御。
  /*
    例：{"display":{ "cols":6, "sm":6, "md":4 }}
  */
  "display": {},
  # 選択項目取得。固定値。
  /*
　{"options":{ "items":[
        {
            "title":"男性",
            "value":"male"
        },
        {
            "title":"女性",
            "value":"female"
        }
       ]
     }
   }
 */
  "options": {},
  # 項目のplaceholder
  "placeholder": "文字",
  # v-select, v-combboxなどの一覧選択
  /*
    "items": [
        {
            "label": "公共交通機関（電車・バス）",
            "value": "1"
        },
        {
            "label": "自家用車（マイカー）",
            "value": "2"
        },
        {
            "label": "バイク",
            "value": "3"
        },
        {
            "label": "自転車",
            "value": "4"
        },
        {
            "label": "徒歩・その他",
            "value": "5"
        }
    ],
  */
  "items": [ ],
  # 入力などにヘルプする情報
  "help": "スタッフの氏名を入力してください",
  # 条件付き表示・条件付き有効化用です。  
  /*
    "dependsOn": {
    "field": "scope_type",
    "operator": "=",
    "value": "custom"
    }
  */
  "dependsOn": null,
  # 保存前変換です。
  /**
    "normalize": {
        "trim": true,
        "toHalfWidth": true,
        "dateFormat": "YYYY-MM-DD"
    }
  */
  "normalize": {
    "trim": true
  },
  # selectアイテムがマスタデータから選択するように
  "masterKey":'roles',
}
```
