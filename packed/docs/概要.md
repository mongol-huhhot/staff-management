##  1. 応募ステータス

```text

interview_scheduled    面談予定
interviewed            面談済み
offer_planned          採用意向
rejected               不採用

```

## 2. 契約ステータス

```text
not_created            未作成
draft                  契約下書き
prepared               契約作成済み
approval_pending       管理者承認待ち
approved               管理者承認済み
sent_to_candidate      本人送付済み
received               本人受領済み
candidate_confirming   本人確認中
candidate_agreed       本人同意済み
signed                 署名済み

```
